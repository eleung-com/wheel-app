# wheel-app

Options wheel strategy tracker (React + Vite).

**All source lives in [`wheel-app/`](wheel-app/)** — see its [README](wheel-app/README.md) for
commands, project structure, and setup.

- Product spec / roadmap: [`wheel-app/docs/PRD.md`](wheel-app/docs/PRD.md)
- Cloudflare Worker proxy source: [`wheel-app/worker/`](wheel-app/worker/)
- Deployed at: https://eleung-com.github.io/wheel-app/ (built to the `gh-pages` branch)

```bash
cd wheel-app
npm install
npm run dev    # http://localhost:5173
```

> (`src/`, `index.html`, …). It was removed on 2026-07-10 — `wheel-app/` is the
> only source of truth.
